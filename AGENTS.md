# AGENTS.md

## Contexto do Projeto

Trabalho 1 da disciplina **Projeto e Análise de Algoritmos**.

Objetivo principal: implementar, testar, comparar e analisar experimentalmente dois algoritmos de ordenação com diferentes complexidades.

## Tema Definido

* Algoritmo quadrático: **Comb Sort**
* Algoritmo O(n log n): **Merge Sort Natural**

## Escopo de Desenvolvimento

1. Implementação completa dos dois algoritmos.
2. Garantia de corretude das implementações.
3. Análise de complexidade assintótica.
4. Execução de experimentos comparativos.
5. Geração de dados para relatório técnico.
6. Apoio à apresentação final.

## Requisitos Experimentais

Executar testes com vetores de diferentes tamanhos nos cenários:

* Vetores ordenados
* Vetores inversamente ordenados
* Vetores aleatórios

Para vetores aleatórios:

* Realizar múltiplas execuções por tamanho (ex.: 10 repetições).
* Consolidar resultados por média ou metodologia equivalente justificada.

## Métricas Esperadas

Coletar uma ou mais métricas:

* Tempo de execução
* Número de comparações
* Número de trocas/movimentações
* Outras métricas relevantes

## Entregáveis Técnicos

### Código

* Implementações organizadas e reproduzíveis.
* Scripts de benchmark/testes.
* Geração de entradas.

### Relatório

Deve permitir produzir:

* Introdução resumida.
* Objetivos.
* Descrição dos algoritmos.
* Prova de corretude.
* Análise teórica (melhor e pior caso).
* Metodologia experimental.
* Tabelas de resultados.
* Gráficos.
* Discussão comparativa.
* Conclusão.

### Apresentação

Resumo dos algoritmos, metodologia, resultados e conclusão.

## Diretrizes do Projeto

Priorizar:

* Implementação em código.
* Otimização e depuração.
* Instrumentação de métricas.
* Benchmarking justo entre algoritmos.
* Visualização de dados.
* Explicações teóricas.
* Escrita técnica do relatório.

## Critérios Importantes

* Comparações devem ser reproduzíveis.
* Mesmo conjunto de entradas para ambos os algoritmos.
* Separar análise teórica de análise experimental.
* Destacar diferenças entre comportamento esperado e observado.
